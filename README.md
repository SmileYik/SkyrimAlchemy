## SkyrimAlchemy

### how to build.

You need Qt5 and CMake 2.8.11 or higher

-- Build instructions --

``` shell
git clone https://github.com/SmileYik/SkyrimAlchemy.git
cd SkyrimAlchemy
mkdir build && cd build
cmake ../
make

cd SkyrimAlchemy
./skyrimalchemy
```
